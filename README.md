## Java Syllabus

- Core Java:
    - Data types,
    - Arrays,
    - Variables types,
    - Operators & Assignments,
    - Loops,
    - Access Modifiers,
    - Declarations & Abstraction,
    - Inheritance,
    - Polymorphism,
    - Encapsulation,
    - Constructor,
    - Interface,
    - Overloading,
    - Overriding,
    - Coupling,
    - Type Casting,
    - Singleton Class,
    - Exception Handling,
    - Multithreading,
    - Multithreading Enhancement,
    - Inner Class,
    - java.lang.package,
    - File IO,
    - Serialization,
    - Collections,
    - Concurrent Collections,
    - Garbage Collections,
    - Internationalization,
    - Enum,
    - Assertions,
    - JVM Arch,
    - Collection, Map, Sorting, Cursor, UtilityClasses:
        - List
            - ArrayList,
            - LinkedList,
            - Vector,
            - Stack,
        - Set
            - HashSet,
            - LinkedHashSet,
            - TreeSet,
        - Map
            - HashMap,
            - LinkedHashMap,
            - IdentityHashMap,
            - WeakHashMap,
            - TreeMap,
            - Hashtable,
            - Properties,
        - Sorting:
            - Comparable,
            - Comparator,
        - Cursor:
            - Enumeration,
            - Iterator,
            - ListIterator,
        - Things to remember while using Collection & Map:
            - Underline data structure,
            - Does implement Serializable, Cloneable, RandomAccess?
            - Duplicates are allowed or not,
            - Insertion order is required or not,
            - Using homogeneous or heterogeneous,
            - Required default or customised sorting order,
            - Null allowed or not,
            - Its constructors,
            - Synchronized or non synchronized (required Thread safety or not?),
            - Does frequent operation a search operation?
